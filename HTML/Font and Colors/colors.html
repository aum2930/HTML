Color can be replaced in...

  1. color keywords (Eg: red, blue...etc)
  2. hex color code (#GGGTE)
  3. RGB (RED: 22(value), GREEN: 04, BLUE: 66)
  4. RGBA (RED: 22(value), GREEN: 04, BLUE: 66, ALPHA (value for opacity of the Particular color)
  5. HSL hsl(value,value,value) (used for the saturation light)
